## 🏗️ Development Setup

Agensic is built with **Python 3.12+** and **Rust**.

```bash
# Fork the repository on GitHub first, then clone your fork
# Replace YOUR_USERNAME with your GitHub username
git clone https://github.com/YOUR_USERNAME/agensic.git
cd agensic

# Add the original repository as 'upstream' to keep your fork in sync
git remote add upstream https://github.com/Alex188dot/agensic.git

# Install Agensic
bash ./install.sh
```

## 📬 Pull Request Guidelines

When submitting a PR, please follow these guidelines:

- **Keep changes focused:** One feature or bug fix per PR
- **Use a feature branch:** Do not work directly on `main`; create a branch such as `feat/my-new-feature` or `fix/issue-123`
- **Ensure tests pass:** Run the test suite before submitting
- **Update documentation:** If you change a feature, update the README or relevant docs
- **Small is better:** We prefer small, composable PRs over large, monolithic ones
